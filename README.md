## 🌐 Portfolio Website

This is a simple and responsive personal portfolio website built using **HTML** and **CSS**. It showcases basic web development skills with a clean layout and easy navigation.

### 📄 Pages Included

* **Home Page** – Introduces the portfolio with a brief overview and welcoming design.
* **Contact Page** – Provides a way for visitors to get in touch.

### ✨ Features

* Responsive design for different screen sizes
* Clean and minimal UI
* Easy navigation between pages
* Structured layout using semantic HTML

### 🛠️ Technologies Used

* HTML5
* CSS3

### 🚀 Purpose

This project was created as a beginner-friendly portfolio to practice and demonstrate fundamental front-end development skills.
